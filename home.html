Para o desenvolvimento deste sistema foi utilizado a linguagem Java com os frameworks Maven, Hibernate e Jersey (para criação de API's Rest).

Para o desenvolvimento do front-end foi utilizado AngularJS e Bootstrap.

Foi utilizando também os gerenciadores de pacotes para o front-end que são Bower e NodeJS.
